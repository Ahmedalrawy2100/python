# Task Manager

My final project after completing an 8-day Python challenge.

## Description

Task Manager is a simple command-line application that helps users manage their daily tasks.

Users can:

- Add new tasks
- Show all tasks
- Mark tasks as completed
- Exit the program

Each task contains:

- Title
- Status (Pending / Done)

---

## Features

1. Add Task
2. Show Tasks
3. Mark Task As Done
4. Exit Program
